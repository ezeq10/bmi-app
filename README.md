## Body Mass Index APP ##

Body Mass Index APP, built on React on FrontEnd and NodeJS on Backend.


## Build Setup

On Server:

```
$ cd bmi-app/server

$ yarn
```

Also you have to create an .env (on server root directory) file having:

```
PORT=XXX

DB_HOST=XXX
DB_PORT=XXX
DB_NAME=XXX
```

On Client:

```
$ cd bmi-app/client

$ yarn
```

## How to use

After installing dependencies, to run client:

```
$ cd bmi-app/client

$ npm start
```

To run server:

```
$ cd bmi-app/server

$ npm start
